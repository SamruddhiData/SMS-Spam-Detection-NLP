📖 Project Overview

This project is an SMS Spam Detection system built using Natural Language Processing (NLP) and Machine Learning. It classifies messages as spam or ham (non-spam).

⚙️ Technologies Used
Python
Pandas, NumPy
NLTK
Scikit-learn
Matplotlib, Seaborn
🔍 Features
Text preprocessing (cleaning, stopwords removal, stemming)
TF-IDF feature extraction
Naive Bayes model
Confusion matrix & visualization
Real-time message prediction
📊 Model Performance
Accuracy: 97%
High precision for spam detection
🚀 How to Run
pip install -r requirements.txt
jupyter notebook

Input:

Congratulations! You won a free iPhone
